Efficient agent recipe

איך להסביר את ארבעת הכרטיסים

Model routing
לא כל שלב צריך premium reasoning. lookup, classification או ranking יכולים לרוץ עם כלי, לוגיקה, או מודל קטן.
Tool grounding
במקום שהמודל ינחש, ה־Agent פונה ל־docs, data, tickets, CRM או MCP tools.
Context redirection
מעבירים בין השלבים distilled state ולא raw transcript, כל המיילים, או כל היסטוריית השיחה.
Eval gates
החיסכון נחשב רק אם האיכות נשארת מעל threshold. המסר: cheap-but-bad הוא לא ניצחון.
